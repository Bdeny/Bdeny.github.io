前言

解决思路

  假设IP地址格式为：a.b.c.d
  由观察可知：每个数字的范围在0~255之间，灭个字段都是模256的余数，则可以将所有字段按照所在位置乘以256的对应次幂。
  由观察可得公式：(a256^3) + (b256^2) + (c256^1) + (d256^0)
这样我们就可以得到一个长整数。每个IP地址对应一个长整数，只要将IP地址与对应求得的长整数用链表保存起来。这样我们就将问题转换为基数排序，结果的排序即为索求排序，只需用指针或者下标等方法将两者对应。
